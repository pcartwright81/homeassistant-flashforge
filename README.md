# Flashforge 3D Printer Integration

This is a Home Assistant integration for FlashForge 3D printers using the flashforge-python-api.

## Installation

1. Download this integration to your Home Assistant custom_components folder.
2. Restart Home Assistant.
3. Add the FlashForge integration via the Home Assistant UI.

## Configuration

Configure your printer's host, username, and password in the integration setup.

## Features

- Monitor printer status
- Control printer actions
